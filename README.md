# Pet Insight

Projeto acadêmico desenvolvido em grupo com foco em desenvolvimento web.

O **Pet Insight** é uma plataforma voltada para informações sobre animais de estimação, cuidados essenciais, curiosidades, campanhas de doação e castração, além de funcionalidades de compra e gerenciamento de produtos para pets.

---

## Funcionalidades

- Sistema de login e cadastro de usuários
- Perfil do usuário e funcionário
- Cadastro, edição e visualização de produtos
- Seção de compras
- Inserção de imagens de produtos
- Conteúdos informativos sobre cuidados e curiosidades de pets

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Composer

---

## Estrutura do Projeto

- `app/` – lógica da aplicação
- `config/` – configurações do sistema
- `public/` – arquivos públicos (HTML, CSS, JS)
- `sql/` – scripts do banco de dados
- `vendor/` – dependências gerenciadas pelo Composer

---

## Projeto Acadêmico

Projeto desenvolvido em grupo como Trabalho de Conclusão de Curso (TCC), com o objetivo de aplicar conhecimentos em desenvolvimento web, banco de dados e lógica de programação.
